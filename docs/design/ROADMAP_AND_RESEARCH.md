# 路线图与研究方向

## 范围

本文件汇总战略方向、阶段计划与研究假设。

来源文档：

- `docs/AGI_VISION.md`
- `docs/notes/project_snapshot_daerwen3.5.md`

## 稳定战略方向

- 以第一性生态底座作为当前主线。
- 在大功能扩展前，优先提升稳健性与可复现性。
- 控制层定位为实验策略层，不替代世界规律层。

## 近期优先级

1. 完成边界安全与模式隔离硬化。
2. 固化 benchmark 协议与 seed 复现实验。
3. 优先优化核心热点循环性能。
4. 明确 warmup 与 measured 指标语义。

## 中期优先级

- 扩展化学与能量循环真实性，
- 提升状态聚合与慢变量记忆信号，
- 开展受控的主动推理/世界模型实验。

## 长期研究主题

- 双进程架构（潜意识生态 + 表意识语言规划），
- 持续学习与长时程自适应，
- 与可证伪实验一致的涌现指标体系。
