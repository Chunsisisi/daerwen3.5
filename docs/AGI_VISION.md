# DAERWEN AGI 涌现路线图

文档状态：`REFERENCE`

> 目标：不是重写一个 CNN/Transformer，而是构建一个真实、可控、可持续的世界，让智能从物理-化学-基因体系中自发增长，并与现代主动推理/持续学习技术对接，最大化 AGI 涌现的概率。

---

## 1. 当前理念概述

1. **最小预设 + 最大计算可行性**  
   - 世界只遵守能量守恒、扩散、化学反应、基因复制等最低层规则。  
   - 不硬编码“行为”，所有复杂性都必须从生态自组织里涌现。  
   - 通过 `ExternalInput` 注入能量/扰动/参数，控制系统只改变环境，不直接操纵个体行为。

2. **生态层是“大脑的物理底座”**  
   - 粒子、化学场、DNA 相互作用构成一个可持续的“活体世界”。  
   - 这个底座提供了真正的开放式动态系统，比在向量空间里训练 CNN 更接近真实智能所需要的物理约束。

3. **输入/输出统一抽象**  
   - 对外暴露 `apply_external_input()` / `get_system_output()`；任何控制器、脚本、测试都用同一套接口与世界交互。  
   - 这保证底层世界保持简单、纯粹，上层逻辑可以不断迭代而不破坏世界本身。

---

## 2. “最大化 AGI 涌现概率” 的路径

### 2.1 生态底座：真实世界引擎
- 持续强化 `ecology_2d` 的物理真实性与可塑性（更多化学、不同能量源、空间/结构多样性）。  
- 保证输入/输出接口稳定，便于外部控制器实验各种策略。

### 2.2 状态聚合器：把世界变成信息表征
- 设计 `State Aggregator`：把 `SystemOutput` 压缩为多分辨率张量/图/约束集合，可被上层模型学习。  
- 作用：等价于“感知层”，帮助控制器理解世界状态，而不引入额外的手工语义。

### 2.3 主动推理/预测控制层
- 用现代世界模型（主动推理、自由能最小化、Dreamer/PlaNet 等）的思想，对 `State Aggregator` 输出来做预测、规划：  
  - 给定状态 `s_t`，预测未来 `s_{t+1}`；  
  - 生成新的 `ExternalInput`（化学脉冲、灾难、参数调节 …），以最小化预测误差或实现多目标指标。  
- 这层负责“想办法让生态更加多样、稳定、向高阶行为演化”，是智能的核心驱动力。

### 2.4 记忆与元学习
- 在控制器中加入慢变量/约束记忆，记录输入与生态响应之间的长期关系（相当于“经验数据库”）。  
- 结合元学习/贝叶斯优化，让控制器能不断改进自己的策略，逐步形成“自我反思”能力。  
- 生态层仍保持无记忆/无脚本，记忆体现在控制层如何持久使用过去的经验作用于未来。  
- DNA、化学场残余、能量分布等慢变量可视作生态内部的“潜意识记忆层”，控制器的显性记忆模块只负责读取、组织与放大这些信号。

### 2.5 层间数据流与接口
- **生态 → 状态聚合器**：`SystemOutput` 输出粒子、化学场、统计指标的固定 schema；聚合器执行多尺度栅格化、时序堆叠与降采样。  
- **聚合器 → 控制器**：提供 `(grid, stats, emergence, history_buffer)` 结构，让控制器同时看到实时状态与最近 N 步轨迹。  
- **控制器 → 生态**：策略统一封装为 `ExternalInput`/WebSocket 消息，可组合 `chemical_pulse + parameter_adjust` 等多通道输入。  
- **时间节奏**：引擎以毫秒级步长运行，控制器可每 20~100 步聚合一次，形成“自由演化 + 周期干预”的稳定节奏。  
- **观测钩子**：聚合器输出中间张量与统计日志，供测试、可视化、上层主动推理模块共享，避免重复解析底层 JSON。

## 3. 最小可行系统 (Minimal Viable System, MVS)

1. **Engine（现有）**  
   - `ecology_2d/core/server/web` + 可调输入。  
   - 能运行长时间，并提供结构化输出。

2. **Controller Stub**  
   - 一个脚本/服务，周期性地读 `SystemOutput` → 生成 `ExternalInput` → 注入世界。  
   - 初版可以是简单规则（如基于涌现得分调节突变率），验证闭环可行。

3. **实验/测试脚本**  
   - 改造 `comprehensive_test_suite.py` 使用新接口；保证所有实验都走“输入→生态→输出”的路径。  
   - 添加 `docs/api_reference.md`，记录输入/输出 JSON schema。

4. **迭代路线**  
   - `State Aggregator`（从 SystemOutput 中提取多尺度特征）  
   - 预测控制器（主动推理/世界模型）  
   - 记忆/元认知模块（长期自适应策略）

---

## 4. 为什么这条路线更接近 AGI

1. **具身与真实**：智能在真实物理约束下演化，而非抽象的参数空间，具备开放性。  
2. **可扩展的闭环**：统一的输入/输出接口，让任何控制策略都可以接入，便于快速试错、迭代。  
3. **与现代技术结合**：引入主动推理/世界模型/元学习等先进理念到真实生态，形成独特的智能生成路径。  
4. **自组织而非脚本**：我们不直接编写“卷积核”或“规则”，而是培养一个能被高层控制器塑形的复杂系统，真正实现“培育”而不是“训练一张网”。

## 6. 生态优化路线图

### 6.1 近期（1~2 个迭代）
- **化学多样性**：扩展 `n_chemical_species` 支持的反应矩阵，加入可逆反应与局部能量源沉降。  
- **能量循环**：建立 ATP → heat → soil 的循环链，让输入/输出能量可追踪、可守恒。  
- **DNA 表达路径**：让 DNA 决定分子装配/代谢优先级，而非直接映射行为，使“潜意识记忆”能在生态层沉淀。  
- **观测指标**：在 `SystemOutput.emergence` 中新增空间熵、能量梯度、化学势差等指标，辅助控制器理解世界。  
- **接口清单**：为每次生态改动列出将新增/修改的输出字段，保持 `ExternalInput/SystemOutput` schema 稳定。

### 6.2 中期（3~5 个迭代）
- **多层化学场**：支持 2.5D（地表/地下/气候）的场层，允许粒子在层间交换资源、形成复杂梯度。  
- **基因装配与遗传干预**：实现基因片段重组、横向转移、嵌入式遗传开关，引入更丰富的演化机制。  
- **长时记忆信号**：在化学场中加入缓慢扩散/衰减的“记忆分子”，供控制器读取生态历史。  
- **生态可塑性研究**：为关键参数（突变率、扩散率、能量注入）设计可在线调整的安全范围，方便主动推理层进行策略探索。

---

这样，我们不仅忠实于“最小预设 + 最大涌现”的理念，还能以现代连续学习、主动推理的方式向高级智能逼近，最大化 AGI 涌现的概率。
